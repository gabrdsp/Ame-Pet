## Descrição do Projeto

O projeto é um sistema de gerenciamento de petshops que integra todas as funcionalidades basicas para um estabelecimento veterinário e de cuidados com animais.
O sistema apresenta uma arquitetura modular e orientada a objetos, que oferece opções de atendimentos clinicos, serviçoes gerais, venda de itens de petshop, compra e adoção de pets.

## Principais funcionalidades:

### Gestão de pessoas
O sistema apresenta uma aba para cadastro e listagem de funcionários, que são os responsáveis por operar o sistema;
Aba de cadastro de clientes.

### Gerenciamento de Pets
Cadastro de animais associados a clientes especificos (seus donos e/ou responsáveis);
Listagem de pets disponíveis para compra e adoção.

### Serviços veterinários e gerais
São ofertados serviços básicos como banho e tosa;
Procedimentos mais especificos, como: consultas médicas, exames e vacinas;
E por fim, serviços mais complexos e delicados, como internações e cirurgias.

### Loja
O sistema conta com uma loja que oferta certos tipos de produtos para animais, como rações, brinquedos e demais itens de uso animal.

### Adoção e venda de pets
Aba especifica para compra e adoção de pets, com formulários para preenchimento com validação de informações, e um sistema de pagamento com descontos e taxas automáticas.

### Relatórios e buscas
O sistema gera um relatório para diversas ações, que vão de compra de itens, até realização de serviços e compra/adoção de pets.
Além disso, pode-se realizar diversas consultas para verificar o histórico de clientes, pets, alem de compra e venda de itens.

## Tutorial de como baixar e executar o projeto

Para baixar o projeto, abra o github na página do repositório do projeto, na aba code, deve-se copiar o link HTTPS disponível.
Após isso, deeve-se criar uma pasta em seu computador, e abrir ela como novo folder em seu VS Code, em seguida, deve-se abrir o terminal e inserir o seguinte comando:
git clone <link do reposítório que foi copiado> e dar enter.
Espere alguns momentos enquanto o projeto é adicionado ao seu computador, depois procure pela File Main.java e execute ela.

## Executando dentro do programa
Com o programa aberto, deve ser cadastrado um funcionário inicial e realizar o login no sistema com seu usuário e senha, após isso, todas as funcionalidades estarão habilitadas para uso.
